# 2. 知识点

## 2.1 变量类型

### 2.1.1 值类型与引用类型

引用类型：对象、数组、函数

引用类型的特点：可以无限追加属性，为了减少内存使用，所以要被设计为引用类型

值类型：普通变量

### 2.2.2 typeof运算符

![](https://ws1.sinaimg.cn/large/a71efaafly1g2rbf7jxk7j20fb07yacd.jpg)

null：是一个空指针，并未有指向任何一个实际的类型，但他的typeof是object

typeof无法区分引用类型的数据，除了函数类型外
